# The Tablet Protocol

An open-source field manual for community-scale ecological stewardship.

We decoded a set of ancient symbols not as myth, but as a schematic for how humans function as living transducers with land and water. This repo contains a simple, 21-minute daily practice any group of 3 to 9 people can use on a school garden, stream bank, or vacant lot.

**No equipment. No cost. Just attention, geometry, and time.**

## Quick Start

1. Choose a small site you already tend and have permission to use
2. Mark a 3-meter triangle with three stones
3. Gather 3 to 9 people at the same time each day
4. Follow the three practices: Spiral (5 min), Jagged Line (2 min), Living Connection (14 min)
5. Record one photo and one sentence for 30 days

Full instructions are in the manual.

## For Schools
- Fits place-based STEM, ecology, and social-emotional learning
- Use as a 30-day citizen science project
- Students learn observation, data collection, and group coherence

## For Watershed Groups
- Add as a low-cost coherence layer before planting or monitoring
- Builds volunteer cohesion
- Simple metrics: water clarity, sprouting, soil moisture
